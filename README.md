## Run Locally

Clone the project

```bash
  git clone https://github.com/williamscch/williamscch-portfolio
```

Go to the project directory

```bash
  cd williamscch-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
